deduceVarValue

Deduce value from an extractor string provided an optional this element The extractor string is a string containing parts spearated by the @ character

Part 1 - One of the following,
			id - Fetch element by eleid and fetch value - needs 3 parts
			this - Use the obj element provided and fetch value - needs 2 parts
			name - Fetch element by elename and fetch value for the first matched element - needs 3 parts
			parent - Fetch elements parent and fetch value for the that element - needs 3 parts
			class - Fetch element by classname and fetch value for the first matched element - needs 3 parts
			selector - Fetch element by jquery jselector and fetch value for the first matched element - needs 3 parts
			constant:{undefined|null|true|false}
			var - get var value - needs 2 parts
			gvar - get global variable value - needs 2 parts
			data - get property value from data object - needs 2 parts
			route - get route fragment value - needs 2 parts
			form - get form element value - needs 2 parts
			object - get value from object - needs 2 parts
Part 2 - One of the following,
			value, valuetrim, text, texttrim, html, htmltrim, a:{attr-name}
Part 3 - Element id, class, name or selector

Examples provided below for reference,

constant:undefined
var@someVar
gvar@someGlobalVar
data@proertyName
route@arg1
form@someEle
object@someProperty
id Extractor (Fetch element by eleid and fetch value)
     id@value@eleid
     id@text@eleid
     id@html@eleid
     id@a:attribute-name@eleid
this Extractor (Use the obj element provided and fetch value)
     this@value
     this@text
     this@html
     this@a:attribute-name
name Extractor (Fetch element by elename and fetch value for the first matched element)
     name@value@elename
     name@text@elename
     name@html@elename
     name@a:attribute-name@elename
parent Extractor (Fetch elements parent and fetch value for the that element)
     parent@value@elename
     parent@text@elename
     parent@html@elename
     parent@a:attribute-name@elename
class Extractor (Fetch element by classname and fetch value for the first matched element)
     class@value@classname
     class@text@classname
     class@html@classname
     class@a:attribute-name@classname
selector Extractor (Fetch element by jquery jselector and fetch value for the first matched element)
     selector@value@jselector
     selector@text@jselector
     selector@html@jselector
     selector@a:attribute-name@jselector
group Extractor (Fetch element in a group of elements with count 'numelements' by element id prefix 'eleidprefix' or selector having class 'matchingclass' 
or having attribute 'macthingattribute' and fetch value according to extractor position)
 By Id prefix and class match
     group:numelements:class:matchingclass:id@value@eleidprefix
     group:numelements:class:matchingclass:id@text@eleidprefix
     group:numelements:class:matchingclass:id@html@eleidprefix
     group:numelements:class:matchingclass:id@a:attribute-name@eleidprefix
 By jquery Selector and class match
     group:numelements:class:matchingclass:@value@jselector
     group:numelements:class:matchingclass:@text@jselector
     group:numelements:class:matchingclass:@html@jselector
     group:numelements:class:matchingclass:@a:attribute-name@jselector
 By Id prefix and attribute exists
     group:numelements:attr:matchingattr:id@value@eleidprefix
     group:numelements:attr:matchingattr:id@text@eleidprefix
     group:numelements:attr:matchingattr:id@html@eleidprefix
     group:numelements:attr:matchingattr:id@a:attribute-name@eleidprefix
 By jquery Selector and attribute exists
     group:numelements:attr:matchingattr:@value@jselector
     group:numelements:attr:matchingattr:@text@jselector
     group:numelements:attr:matchingattr:@html@jselector
     group:numelements:attr:matchingattr:@a:attribute-name@jselector
 By Id prefix and attribute match
     group:numelements:attr:matchingattr:id:attrvalue@value@eleidprefix
     group:numelements:attr:matchingattr:id:attrvalue@text@eleidprefix
     group:numelements:attr:matchingattr:id:attrvalue@html@eleidprefix
     group:numelements:attr:matchingattr:id:attrvalue@a:attribute-name@eleidprefix
 By jquery Selector and attribute match

validateCondition

Given a condition string and an element object verify the condition and return true/false whether the condition holds true or not Condtions are a pipe separated string which has 3 parts and a value extractor part separated by a colon

eg |=|ABC:this@value or |!=|BCD:id@value@eid
Part 1
		len or length (absent) for non lenght based checks
		regex, Part 3 should have regex string
Part 2 - condition operator (=,!=,<,<=,>,>=,!n,n!u,u) which includes null and undefined checks
Part 3 - the value to be checked
Part after ":" (colon)
	 	value to be compared against @see deduceVarValue for the spec for the same
